Install and Update eMoflon

This page provides installation information for eMoflon. eMoflon is currently composed of two parts: eMoflon IDE Eclipse plugin Enterprise Architect addin (only required for visual syntax support) If you are completely new to eMoflon we recommend working through Opens internal link in current windowour handbook first!

Versions of eMoflon

Note 1: Our EA extension and Eclipse plugin cannot be mixed arbitrarily. If you encounter any problems when exporting your metamodels from EA, first check whether you have the appropriate version of the EA addin installed.

Note 2: If you encounter problems, pleas consult our issue tracker or drop us a mail at contact(at)emoflon.org. Thanks!

Updating eMoflon

Important: You always need to update both parts of eMoflon.

To update eMoflon, proceed as follows.

  • Use the Eclipse update manager to check for and install updates Deinstall our EA extension via Windows' Control Panel / Programs and Features
  • Re-install the latest version of our EA extension by using one of the URLs above.
